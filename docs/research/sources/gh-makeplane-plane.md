# makeplane/plane — Open-source Jira, Linear, Monday, and ClickUp alternative. Plane is a…

## Summary

makeplane/plane is a large, AGPL-3.0 open-source project management platform positioned as an alternative to Jira, Linear, Monday, and ClickUp. It combines a React/TypeScript frontend with a Django/Python backend and PostgreSQL/Redis, offering work items, cycles, modules, filtered views, AI-enabled pages, and analytics. It matters as a mature reference for issue tracking, sprint-style workflows, and self-hosted product-management UX at significant scale (~50k GitHub stars).

## Key claims

- Plane is an open-source project management tool for issues, cycles (sprints), modules, roadmaps, and team workflows
- Stack is React/TypeScript (Vite, React Router) on the frontend and Django/Python on the backend, with PostgreSQL and Redis
- Core product surfaces include Work Items, Cycles with burn-down charts, Modules, saved Views, Pages with AI and rich text, and Analytics
- Available as managed Plane Cloud or self-hosted via Docker Compose and Kubernetes with instance-admin 'God mode' configuration
- Licensed under GNU Affero General Public License v3.0 (AGPL-3.0)
- Community and contribution channels include GitHub Discussions, a dedicated forum, and structured bug/feature issue templates
- Security disclosures are handled privately via security@plane.so rather than public issues
