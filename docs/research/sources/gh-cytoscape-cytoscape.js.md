# cytoscape/cytoscape.js — Graph theory (network) library for visualisation and analysis

## Summary

Cytoscape.js is a mature, MIT-licensed JavaScript graph-theory library (~11k GitHub stars) for modeling, analyzing, and interactively visualizing relational networks in the browser or Node.js. It pairs a full graph data model with an optional renderer, ships on npm with UMD/ESM/CJS builds, and supports a large extension ecosystem. Originating from University of Toronto research and published in Oxford Bioinformatics (2016, updated 2023), it is widely used for biological networks, social graphs, and general dependency visualization.

## Key claims

- Provides a complete graph-theory model plus an optional interactive renderer for relational data visualization and analysis
- Initializes with a single call: cytoscape({ elements, container }) for browser or Node.js use
- Targets programmers and scientists who need graph modeling for server-side analysis or rich UIs
- Distributed on npm as cytoscape with UMD, ESM, and CJS build targets and documented build/test workflows
- Maintains an extension ecosystem of 70+ plugins for layouts, analysis, and rendering enhancements
- Released under the MIT license with monthly feature releases and weekly patch releases
- Peer-reviewed and citable via Oxford Bioinformatics publications (2016 original, 2023 update)
- Documented with live demos (e.g. Tokyo railway network), tutorials, and a project website at js.cytoscape.org
