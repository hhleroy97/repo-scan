# humanlayer/humanlayer — The best way to get AI coding agents to solve hard problems in complex…

## Summary

humanlayer/humanlayer is an open-source TypeScript project (Apache-2) from the HumanLayer team, now centered on CodeLayer—an IDE for orchestrating AI coding agents (especially Claude Code) in large codebases. It promotes keyboard-first workflows, parallel agent sessions (worktrees, remote workers), and team-scale context engineering. The repo also hosts foundational material on context engineering and 12 Factor Agents; the original HumanLayer SDK docs live in legacy docs.

## Key claims

- CodeLayer is an open-source IDE for orchestrating AI coding agents with battle-tested workflows for hard problems in complex codebases
- Built on Claude Code with keyboard-first, speed-oriented workflows and support for parallel sessions via worktrees and remote cloud workers
- Advanced context engineering is positioned as the way to scale AI-first development across teams without chaotic output
- The team originated the term context engineering (April 2025) and publishes related talks, 12 Factor Agents principles, and the AI That Works podcast
- CodeLayer early access is waitlist-only via npx humanlayer join-waitlist; the HumanLayer SDK remains documented separately as legacy
- HumanLayer offers paid team services (custom workflows, integrations, hands-on engineering) alongside the open-source codebase
