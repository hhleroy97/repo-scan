# seddonym/import-linter — Lint your Python architecture.

## Summary

Import Linter is an open-source Python tool that enforces architectural rules by constraining which modules may import one another. It is published on PyPI, maintained on GitHub, and licensed under BSD 2-Clause. It also offers a browser-based UI for exploring a package’s import graph and dependency structure.

## Key claims

- Allows you to impose constraints on imports between Python modules
- Provides a browser-based user interface for exploring the architecture of any Python package
- Distributed on PyPI as import-linter with documented getting-started guides
- Open source under the BSD 2-Clause license
- Focused on dependency and import structure as a form of architectural linting
- Actively maintained with CI workflows on GitHub
