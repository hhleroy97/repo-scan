## Summary

pypa/sampleproject is the official reference sample Python package maintained by the Python Packaging Authority (PyPA). It exists primarily to support the Python Packaging User Guide's tutorial on packaging and distributing projects, demonstrating how project metadata is defined in pyproject.toml. It is not intended as a full template for development practices such as version control, documentation, or testing.

## Key claims

- Project metadata for a Python package is defined in pyproject.toml, with an example included in this repository
- The README is used as the long description on PyPI and as the project homepage on code-hosting services
- README files may use UTF-8-encoded reStructuredText or Markdown with the appropriate core-metadata description content-type
- Typical README content includes an overview and basic usage examples; a full changelog is generally discouraged
- This sample does not aim to cover best practices for Python project development beyond packaging
